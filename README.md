# world_cup_stats
I will look at the best performer from the world cup based on their performer at the games

This project showcases a data‑driven dashboard system for analyzing and visualizing tournament performance. Using SQL for data management and Python (pandas, Plotly, Streamlit) for analysis and visualization, the repo highlights top performers across different categories:

- Goalkeeper of the Tournament
- Defender of the Tournament
- Top Assister of the Tournament
- Top Scorer of the Tournament
- Best Player of the Tournament
- Team of the Tournament

Each category has its own dedicated dashboard with interactive charts, stats, and visualizations.
